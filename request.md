- **game-poor**

## 목표
Steam, Epic Games Store, Nintendo eShop 등 게임 플랫폼의 할인/이벤트/출시 정보를 매일 밤 9시에 이메일로 받아보는 자동화 시스템 구축

## 핵심 요구사항

### 1. 알림 대상 정보
- 게임 할인 (세일, 특가)
- 무료 게임 이벤트 (Epic 주간 무료 등)
- 신작 출시 정보
- 관심 게임 가격 변동

### 2. 대상 플랫폼
- **Steam** — 1순위
- **Epic Games Store** — 1순위
- **Nintendo eShop** — 2순위
- 추후 플랫폼 추가 가능 (GOG, PlayStation Store 등)

### 3. 개인화
- 각 플랫폼의 찜목록(위시리스트) 기반 알림 (가능한 경우)
- 찜목록 접근 불가 시 직접 지정한 관심 게임 목록 기반
- 관심 게임 목록은 수시로 추가/삭제 가능해야 함

### 4. 알림 방식
- **Claude Code의 schedule 기능** 활용
- 매일 밤 **21:00 (KST)** 실행
- **이메일**로 수신

### 5. 관리 편의성
- 별칭 `game-poor`로 Claude Code에서 관리
- 관심 게임 목록 변경 시 대화로 요청하면 반영
- 플랫폼 추가/제거 시 대화로 요청하면 반영
- 설정 파일 기반으로 유지보수 용이하게 구성
